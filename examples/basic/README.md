# basic

Basic example using `oak-http-proxy` as a proxy middleware.

## How to run this example

Run this example using:

```bash
deno run --allow-net --allow-read ./examples/proxy/index.ts
```

if have the repo cloned locally _OR_

```bash
deno run --allow-net --allow-read https://deno.land/x/oak_http_proxy@2.0.1/examples/basic/index.ts
```

if you don't!
